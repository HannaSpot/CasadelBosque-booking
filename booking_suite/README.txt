SELVATICA DIRECT BOOKING SUITE V1

Thank you for purchasing Selvatica Direct Booking Suite.

WHAT IS INCLUDED
- booking-system.html
- editable-config.js
- setup-guide.pdf
- assets folder
- demo-preview folder
- README.txt

FAST START
1. Open editable-config.js
2. Change your hotel/property name
3. Replace WhatsApp number with your number
4. Edit room names, prices, amenities and blocked dates
5. Open booking-system.html in your browser to test
6. Upload booking-system.html and editable-config.js to your website or embed it

IMPORTANT WHATSAPP FORMAT
Use country code with no plus sign.
Example:
+57 300 000 0000 becomes 573000000000

HOW TO ADD / EDIT ROOMS
Open editable-config.js and copy one room block.
Change:
- id
- name
- type
- price
- capacity
- image
- description
- amenities
- blockedDates

BLOCKED DATES FORMAT
Use:
YYYY-MM-DD

Example:
blockedDates: ["2026-05-10", "2026-05-11"]

IMAGE OPTIONS
You can use:
- Online image URLs
- Images uploaded to your website
- Images inside assets/rooms if your website supports local files
